# Scripts

Scripts for converting from `.ml`/`.mli` to `.re`/`.rei` files.


## Usage:

Without passing arguments
```bash
‚ùØ npx mlxre
1) ml2re
2) mli2rei
3) delete
4) quit
Select the operation: 
```

Or

```sh
npx mlxre ml2re
npx mlxre mli2rei
npx mlxre re2ml
npx mlxre rei2mli
npx mlxre delete
```
